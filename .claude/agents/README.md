# Agentes Especializados - Sistema de Orçamentos

Este diretório contém configurações e prompts para agentes especializados que ajudam no desenvolvimento do sistema.

## Agentes Disponíveis

### 1. Frontend Agent
Especializado em componentes React, interfaces do usuário e estilização.

### 2. Backend/Database Agent
Especializado em Supabase, SQL, migrations e schema do banco.

### 3. Planejamento Agent
Especializado na lógica de negócio de planejamento de produção.

### 4. Debug Agent
Especializado em investigação e correção de bugs.

### 5. Test Agent
Especializado em criação e manutenção de testes.

### 6. Documentation Agent
Especializado em manter documentação atualizada.

## Como Usar

Referencie o agente apropriado ao fazer solicitações, por exemplo:
- "Como frontend agent, crie um componente para..."
- "Como database agent, adicione uma migration para..."
