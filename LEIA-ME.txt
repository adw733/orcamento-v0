💡 DICAS DE USO:

🔄 FLUXO TÍPICO:
1. Abra o sistema: INICIAR.bat
2. Faça modificações no código
3. Vá para aba "📦 Versões"
4. Digite descrição e clique "💾 SALVAR VERSÃO"
5. Para testar: aba "🚀 Projeto" → "🚀 INICIAR SERVIDOR"
6. Para publicar: aba "🌐 Deploy" → "🔍 PREVIEW" ou "🌐 PRODUÇÃO"

🎯 PRIMEIRO USO DA VERCEL:
1. Aba "🌐 Deploy"
2. Clique "📦 INSTALAR CLI" (automático)
3. Clique "🔑 FAZER LOGIN" (abre navegador)
4. Faça login na Vercel
5. Agora pode fazer deploy!

🔀 TRABALHANDO COM FORKS:
- Use forks para testar ideias sem afetar o projeto principal
- Crie fork, trabalhe nele, depois volte para o principal
- Cada fork é uma "versão paralela" do projeto

🆘 PROBLEMAS COMUNS:

❓ "Python não encontrado"
   → Instale Python: https://python.org/downloads

❓ "Arquivos não encontrados"  
   → Certifique-se que está na pasta correta do projeto

❓ "Erro ao iniciar servidor"
   → Instale dependências: npm install

❓ "Vercel CLI não funciona"
   → Use o botão "📦 INSTALAR CLI" no sistema
